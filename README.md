##初次使用
| 命令作用                             | 命令代码                               |
| :------------                        |:---------------                        |
| 进入本地文件夹                       | cd [路径]                              |
| 初始化本地git仓库                    | git init                               |
| 添加文件                             | git add *                              |
| 查看添加的不同的文件状态             | git status                             |
| 提交文件变化                         | git commit -m “[提交tag]”              |
| 提交到远程仓库                       | git push origin master                 |
